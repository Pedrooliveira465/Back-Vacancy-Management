## Vacancy Management
Vacancy Management is a project to manage vacancies, making it easier for users to find and apply for vacancies, and for companies to register their vacancies.


## Technology 

Here are the technologies used in this project.

* Spring Boot
* Spring Security
* JUnit
* AWS
* CI/CD
* JPA
* Docker 
* Docker Compose
* PostgreSQL
* Swagger
* SonarQube
* Actuator
* Prometheus
* Grafana

## Services Used

* Github
* AWS

## Getting started

* Dependency
  - Docker
  - Docker Compose
  
* To run the project:
> docker-compose up -d
